# 问卷后台使用说明

## 现在在哪里看记录

- 打开 `admin.html` 可以看本机浏览器保存的测试记录。
- 只有接入 Google 表格后台后，别人在手机上提交的记录才会集中保存。

## 接入在线后台

1. 新建一个 Google 表格。
2. 在表格里打开“扩展程序” -> “Apps Script”。
3. 把 `google-apps-script/Code.gs` 的内容复制进去。
4. 把 `ADMIN_TOKEN` 改成你的后台密码。
5. 把 `ANSWER_KEY` 填好。
6. 部署为 Web App，访问权限选择“任何人”。
7. 复制 Web App 地址，填到 `config.js` 的 `backendUrl`。
8. 重新上传到 GitHub 后，手机提交记录就会进入 Google 表格。

## 答案表格式

第 1 道选择题对应 `q3`，第 52 道选择题对应 `q54`。

示例：

```js
const ANSWER_KEY = {
  q3: '4',
  q4: '3',
  q5: '2'
};
```
